ファイルマネージャ枠の現在のフォルダのパスを取得(秀丸エディタ 8.85以上)

概要

フォルダツリーとファイル一覧の現在のフォルダのパスを取得します。

秀丸本体のカレントフォルダとは異なるので注意してください。
あくまでもファイルマネージャ枠のカレントフォルダです。

フォルダツリーとファイル一覧の現在のフォルダのパスを取得

非同期のメソッド中も含め、常時取得することが出来ます。

string currentdir = hm.ExplorerPane.GetCurrentDir()
