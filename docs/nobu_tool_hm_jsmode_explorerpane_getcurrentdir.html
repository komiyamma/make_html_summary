ファイルマネージャ枠の現在のフォルダのパスを取得

getCurrentDir

フォルダツリーとファイル一覧の現在のフォルダのパスを取得します。
秀丸本体のカレントフォルダとは異なるので注意してください。
あくまでもファイルマネージャ枠のカレントフォルダです。

ファイルマネージャ枠の現在のフォルダのパスを取得

namespace ExplorerPane {
    getCurrentDir(): string;
}

ファイルマネージャ枠の現在のフォルダのパスを返す。
失敗した場合、空文字を返す。

aaa.js 
debuginfo(2);
var curdir = ExplorerPane.getCurrentDir();
console.log(curdir);
